# OIBSIP_PythonProgramming_Task5

## Objective
Develop a browser‑based chat application using **Flask** and **SocketIO** that enables real‑time communication between multiple users through a web interface.

## Steps Performed
- Installed required dependencies:
  ```bash
  pip install flask flask-socketio

- Created a backend Python file (Aswathi_Task5.py) using Flask and SocketIO.
- Designed a frontend (chat.html) with HTML, CSS, and JavaScript for the chat interface.
- Implemented WebSocket connections to broadcast messages instantly to all connected clients.
- Tested the application by opening multiple browser tabs and verifying real‑time message updates.
- Captured screenshots of:
   - Server running in terminal
   - Chat interface in browser


## Tools Used
- Python 3.14
- VS Code
- Flask ,Flask‑SocketIO
- HTML, CSS, JavaScript
- VS Code

## Outcome

The application successfully allows multiple users to chat in real‑time via a browser interface, demonstrating integration of Python backend with a web frontend.
