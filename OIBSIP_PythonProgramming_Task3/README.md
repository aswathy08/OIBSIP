# OIBSIP_PythonProgramming_Task3

## Objective
Develop a Python program that generates secure random passwords based on user preferences, ensuring strong password creation for better security.

## Steps Performed
- Collected user input for desired password length.
- Allowed options to include:
  - Uppercase letters
  - Lowercase letters
  - Digits
  - Symbols
- Built a character pool based on user choices.
- Guaranteed at least one character from each selected type.
- Generated a random password using Python’s `random` library.
- Implemented a strength meter to classify the password as Weak, Medium, or Strong.

## Tools Used
- Python 3.14
- VS Code
- Standard Python libraries (`random`, `string`)

## Outcome
The program successfully generates secure random passwords according to user preferences and evaluates their strength.
