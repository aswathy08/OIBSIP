# OIBSIP_PythonProgramming_Task4

## Objective
Develop a Python program that fetches and displays real-time weather information for a given city using the OpenWeatherMap API.

## Steps Performed
- Registered on OpenWeatherMap and obtained an API key.
- Collected user input for the city name.
- Sent a request to the OpenWeatherMap API using the `requests` library.
- Parsed the JSON response to extract:
  - Temperature (°C)
  - Humidity (%)
  - Weather condition (e.g., Clear, Rainy, Cloudy)
- Implemented error handling for invalid city names or API issues.
- Displayed the weather details neatly in the console.

## Tools Used
- Python 3.14
- VS Code
- Libraries: `requests`, `json`

## Outcome
The program successfully fetches live weather data from OpenWeatherMap and displays temperature, humidity, and condition for the entered city.
